## MdButtonComponent
Custom element representing a material design button.

**Kind**: global class  
**Emits**: <code>event:{CustomEvent} md-button-click - Dispatched when the button is clicked.</code>  
**Element**: md-button  
**Cssprop**: <code>String</code> --md-button-color - Color of the button.  
**Cssprop**: <code>String</code> --md-button-background - Background color of the button.  
**Properties**

| Name | Type | Description |
| --- | --- | --- |
| type | <code>String</code> | The type of the button (e.g., "button", "submit"). |
| label | <code>String</code> | The label text of the button. |
| icon | <code>String</code> | The icon content of the button. |
| ui | <code>String</code> | The UI style of the button ("elevated", "filled", "filled-tonal", "outlined"). |
| activated | <code>Boolean</code> | Indicates whether the button is activated. |
