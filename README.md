# Material Design

Material 3 adalah versi terbaru dari sistem desain open-source Google. Dengan Material 3, Anda dapat merancang dan membangun produk yang indah, fungsional, dan mudah digunakan.

## 🌟 Fitur Utama
- **Desain Modern**: Ikuti prinsip Material 3 untuk tampilan yang bersih dan profesional.
- **Responsif**: Sesuai untuk berbagai perangkat dan ukuran layar.
- **Komponen Fleksibel**: Mudah dikustomisasi sesuai kebutuhan.

## 📌 Instalasi
Gunakan perintah berikut untuk mulai menggunakan Material 3 dalam proyek Anda:
```bash
npm install material-3
```
Atau tambahkan langsung dari CDN:
```html
<link rel="stylesheet" href="https://ndiing.github.io/material/dist/material.min.css">
```

## 📖 Panduan Penggunaan
Tambahkan komponen Material 3 dengan struktur berikut:
```html
<button class="md-button">Klik Saya</button>
```
Lebih banyak contoh bisa ditemukan di [dokumentasi resmi](https://ndiing.github.io/material/docs/).
