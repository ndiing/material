# MDTooltipComponent
The `MDTooltipComponent` interface represents a `<md-tooltip>` element in the DOM. Represents a tooltip component.

## Inheritance
`MDSheetComponent`


## Instance properties
This interface also inherits properties from its parent, `MDSheetComponent`. 

name | type | desc
--- | --- | ---
`variants` | `ReadOnly` | Variants of the tooltip component.

## Instance methods
This interface also inherits methods from its parent, `MDSheetComponent`.

name | params | desc
--- | --- | ---
`showModal` | `button`,`options` | Shows the tooltip as a modal.
`show` | `button`,`options` | Shows the tooltip.
`setPlacement` | `button`,`options` | Sets the placement of the tooltip relative to the provided button element.
