# MDSwitchComponent

The `MDSwitchComponent` interface represents a `<md-switch>` element in the DOM.
MDSwitchComponent is a custom switch element with support for ripple effects.



### Events

| Event | Description |
|-------|-------------|
| `MDSwitchComponent#event:onSwitchNativeInput - Fired when the switch&#x27;s native input value changes.` | |
| `MDSwitchComponent#event:onSwitchNativeReset - Fired when the switch&#x27;s native input is reset.` | |




# properties


### Instance properties

| Name | Type | Description |
|------|------|-------------|
| `checked` | (Boolean) | Indicates whether the switch is checked. |
| `defaultChecked` | (Boolean) | The default checked state of the switch. |
| `disabled` | (Boolean) | Indicates whether the switch is disabled. |
| `indeterminate` | (Boolean) | Indicates whether the switch is in an indeterminate state. |
| `value` | (String) | The value associated with the switch. |
| `name` | (String) | The name attribute for the switch. |
| `form` | (Object) | The form element associated with the switch. |
| `type` | (String) | The type of the switch, typically &quot;checkbox&quot;. |
| `icons` | (Array) | An array of icons to be displayed based on the switch state. |
