# MDCardComponent

The `MDCardComponent` interface represents a `<md-card>` element in the DOM.
Custom card component.



### Events

| Event | Description |
|-------|-------------|
| `MDCardComponent#event:onCardIconButtonClick - When an icon button in the card is clicked.` | |
| `MDCardComponent#event:onCardIconClick - When an icon in the card is clicked.` | |
| `MDCardComponent#event:onCardButtonClick - When a button in the card is clicked.` | |
| `MDCardComponent#event:onCardFabClick - When a FAB (Floating Action Button) in the card is clicked.` | |
| `MDCardComponent#event:onCardTextFieldNativeFocus - When a native input in the card gains focus.` | |
| `MDCardComponent#event:onCardTextFieldNativeBlur - When a native input in the card loses focus.` | |
| `MDCardComponent#event:onCardTextFieldNativeInput - When a native input in the card receives input.` | |
| `MDCardComponent#event:onCardTextFieldNativeSearch - When a native input in the card performs a search.` | |
| `MDCardComponent#event:onCardTextFieldNativeInvalid - When a native input in the card becomes invalid.` | |
| `MDCardComponent#event:onCardTextFieldNativeReset - When a native input in the card is reset.` | |




# properties


### Instance properties

| Name | Type | Description |
|------|------|-------------|
| `variant` | (String) | Variant of the card (e.g., &quot;elevated&quot;, &quot;filled&quot;, &quot;outlined&quot;). |
| `leadingActions` | (Array) | Array of leading actions (e.g., icons, buttons) in the card. |
| `label` | (String) | Main label text for the card. |
| `subLabel` | (String) | Sub-label text for the card. |
| `trailingActions` | (Array) | Array of trailing actions (e.g., icons, buttons) in the card. |
| `actions` | (Array) | Array of actions (e.g., buttons) within the card body or footer. |
