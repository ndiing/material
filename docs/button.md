# MDButtonComponent
The `MDButtonComponent` interface represents a `<md-button>` element in the DOM. A custom element for creating various styles of buttons with optional ripple effects.

## Inheritance
`MDComponent`


## Instance properties
This interface also inherits properties from its parent, `MDComponent`. Defines the properties of the element.

name | type | desc
--- | --- | ---
`variant` | `String` | The style variant of the button (e.g., "elevated", "filled", "tonal", "outlined", "icon-right").
`type` | `String` | The type attribute of the button element (default is "button").
`icon` | `String` | The icon to display within the button.
`label` | `String` | The label text to display within the button.
`selected` | `Boolean` | Indicates whether the button is selected.
`disabled` | `Boolean` | Indicates whether the button is disabled.
