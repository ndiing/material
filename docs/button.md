# MDButtonComponent

The `MDButtonComponent` interface represents a `<md-button>` element in the DOM. Button component for Material Design.

## Instance Properties
This interface also inherits properties from its parent, `MDComponent`.

| Name | Type | Description |
| --- | --- | --- |
| variant | `String` | The variant style of the button (e.g., elevated, filled, outlined). |
| type | `String` | The type attribute of the button (default is "button"). |
| icon | `String` | Optional icon to display inside the button. |
| label | `String` | The label or text content of the button. |
| selected | `Boolean` | Indicates if the button is selected. |
| disabled | `Boolean` | Indicates if the button is disabled. |
