Class: MDRippleController

MDRippleController(host, optionsopt)

Represents a controller for Material Design Ripple effect. Extends MDController.

Constructor

new MDRippleController(host, optionsopt)

Creates an instance of MDRippleController.
Parameters:
Name Type Attributes Default Description
host HTMLElement The host element.
options Object <optional>
 {} Additional options for the controller.
Properties:
Name Type Description
container HTMLElement The container element for the ripple effect.
button HTMLElement The button element triggering the ripple effect.
containment boolean Indicates whether the ripple is contained within the button.
inverted boolean Indicates whether the ripple effect is inverted.
fadeout boolean Indicates whether the ripple effect should fade out.
size number | null The size of the ripple effect, or null to calculate dynamically.
centered boolean Indicates whether the ripple effect should be centered.
