## Instance properties
name | type | desc
--- | --- | ---
`requestId` | `ReadOnly` | Represents a controller for managing a progress indicator.

## Instance methods
name | params | desc
--- | --- | ---
`create` |  | Creates a new progress indicator element and appends it to the document body.
`reset` |  | Resets the progress indicator and animation state.
`observe` | `resolve` | Observes the progress of the animation.
`start` | `duration` | Starts the progress animation with a specified duration.
`pause` |  | Pauses the current animation.
`resume` |  | Resumes a paused animation.
`stop` |  | Stops the current animation and resets the progress indicator.
