# MDScrimComponent
The `MDScrimComponent` interface represents a `<md-scrim>` element in the DOM. Represents a scrim component that extends MDComponent.

## Inheritance
`MDComponent`


## Instance properties
This interface also inherits properties from its parent, `MDComponent`. Properties of the MDScrimComponent.

name | type | desc
--- | --- | ---
`open` | `Boolean` | Reflects whether the scrim is open.

## Instance methods
This interface also inherits methods from its parent, `MDComponent`.

name | params | desc
--- | --- | ---
`show` |  | Shows the scrim by setting `open` to true.
`close` |  | Hides the scrim by setting `open` to false.
`toggle` |  | Toggles the scrim visibility.
