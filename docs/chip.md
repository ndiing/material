# MDChipComponent

The `MDChipComponent` interface represents a `<md-chip>` element in the DOM.
MDChipComponent is a class that extends MDComponent and represents a custom chip component.
This component provides various visual and interactive states based on its properties.



### Events

| Event | Description |
|-------|-------------|
| `MDChipComponent#event:onChipActionClick - Fires when the chip action button is clicked.` | |




# properties


### Instance properties

| Name | Type | Description |
|------|------|-------------|
| `variant` | (String) | The variant type of the chip (e.g., &quot;assist&quot;, &quot;filter&quot;, &quot;input&quot;, &quot;suggestion&quot;). |
| `avatar` | (String) | The URL of the avatar image to be displayed in the chip. |
| `icon` | (String) | The icon to be displayed in the chip. |
| `label` | (String) | The text label of the chip. |
| `action` | (String) | The icon for the action button in the chip. |
| `selected` | (Boolean) | Whether the chip is in a selected state. |
| `disabled` | (Boolean) | Whether the chip is disabled. |
