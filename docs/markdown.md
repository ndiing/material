# MDMarkdownComponent

The `MDMarkdownComponent` interface represents a `<md-markdown>` element in the DOM. Custom element for rendering Markdown content.

## Instance Properties
This interface also inherits properties from its parent, `MDComponent`.

| Name | Type | Description |
| --- | --- | --- |
| href | `String` | The URL from which to fetch Markdown content. |
