# MDMarkdownComponent
The `MDMarkdownComponent` interface represents a `<md-markdown>` element in the DOM. A custom element for rendering Markdown content.

## Inheritance
`MDComponent`


## Instance properties
This interface also inherits properties from its parent, `MDComponent`. Defines the properties of the element.

name | type | desc
--- | --- | ---
`href` | `String` | The URL to fetch Markdown content from.
`text` | `String` | The Markdown text to be rendered.
