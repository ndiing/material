# MDRippleController
MDRippleController provides ripple effect handling for UI interactions.

## Constructor
Creates an instance of MDRippleController.

name | type | desc
--- | --- | --- 
host | HTMLElement | The host element where ripple effects are applied.
options | Object | Options for configuring ripple behavior.
options.buttonSelector | String,null | Selector for the button element.
options.centered | Boolean | Whether the ripple effect should be centered.
options.clipped | Boolean | Whether the ripple effect should be clipped.
options.containerSelector | String,null | Selector for the container element.
options.fadeOut | Boolean | Whether the ripple effect should fade out.
options.size | Number,null | Size of the ripple effect.
