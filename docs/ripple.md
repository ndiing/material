## MDRippleController
Controller for managing ripple effects on a host element.

**Kind**: global class  
<a name="new_MDRippleController_new"></a>

### new MDRippleController(host, options)
Constructs a new MDRippleController instance.


| Param | Type | Description |
| --- | --- | --- |
| host | <code>\*</code> | The host element to attach the ripple effect. |
| options | <code>Object</code> | Options for configuring the ripple behavior. |
| options.buttonSelector | <code>String</code> | Selector for the button element within the host. |
| options.centered | <code>Boolean</code> | Whether the ripple should be centered on the click position. |
| options.clipped | <code>Boolean</code> | Whether the ripple effect is clipped within the container. |
| options.containerSelector | <code>String</code> | Selector for the container element within the host. |
| options.fadeOut | <code>Boolean</code> | Whether the ripple effect fades out. |
| options.size | <code>Number</code> | Size of the ripple effect relative to the container size. |
