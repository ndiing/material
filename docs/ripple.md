# MDRippleController

Constructs a new MDRippleController instance.





#### Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `host` | (*) | The host element to attach the ripple effect. |
| `options` | (Object) | Options for configuring the ripple behavior. |
| `options.buttonSelector` | (String) | Selector for the button element within the host. |
| `options.centered` | (Boolean) | Whether the ripple should be centered on the click position. |
| `options.clipped` | (Boolean) | Whether the ripple effect is clipped within the container. |
| `options.containerSelector` | (String) | Selector for the container element within the host. |
| `options.fadeOut` | (Boolean) | Whether the ripple effect fades out. |
| `options.size` | (Number) | Size of the ripple effect relative to the container size. |


# MDRippleController

Controller for managing ripple effects on a host element.
