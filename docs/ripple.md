# MDRippleController
Provides a controller for adding ripple effects to elements.

## Constructor
Initializes the ripple controller for a host element.

name | type | desc
--- | --- | --- 
`host` | `HTMLElement` | Host element to apply ripple effect.
`options` | `Object` | Configuration options for the ripple effect.
`options.buttonSelector` | `string` | Selector for the button element within the host.
`options.centered` | `boolean` | Whether the ripple effect should be centered.
`options.clipped` | `boolean` | Whether the ripple effect should be clipped.
`options.containerSelector` | `string` | Selector for the container element within the host.
`options.fadeOut` | `boolean` | Whether the ripple effect should fade out.
`options.size` | `number` | Size of the ripple effect relative to the container.
