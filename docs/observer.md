

# MDObserver
Defines media query observer for handling changes based on provided list.
This class allows monitoring changes in media queries and triggering a callback function.

## Constructor
Creates an instance of MDObserver.

name | type | desc
--- | --- | --- 
callback | function | Optional callback function triggered on query change.


## Instance methods
name | params | desc
--- | --- | ---
observe | list | Observes changes in media queries based on the provided list.
