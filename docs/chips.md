# MDChipsComponent

The `MDChipsComponent` interface represents a `<md-chips>` element in the DOM.
MDChipsComponent is a class that extends MDComponent and represents a collection of chip components.
This component manages an array of chip data and handles their interactions.



### Events

| Event | Description |
|-------|-------------|
| `MDChipsComponent#event:onChipClick - Fires when a chip in the collection is clicked.` | |




# properties


### Instance properties

| Name | Type | Description |
|------|------|-------------|
| `list` | (Array) | The list of chip data to be rendered. |
| `multiSelection` | (Boolean) | Whether multiple chips can be selected simultaneously. |
