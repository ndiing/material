# MDDateFieldComponent
The `MDDateFieldComponent` interface represents a `md-date-field` element in the DOM. Represents a date field component extending MDTextFieldComponent.

## Inheritance
MDTextFieldComponent


## Instance properties
This interface also inherits properties from its parent, `MDTextFieldComponent`. 

name | type | desc
--- | --- | ---
actions | Read only | Defines the actions associated with the date field.
actions | Read only | Sets the actions associated with the date field.
