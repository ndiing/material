# MDListItemComponent

The `MDListItemComponent` interface represents a `<md-list-item>` element in the DOM.
Represents a custom list item component with various content options such as avatar, thumbnail, icon, and more.







# properties


### Instance properties

| Name | Type | Description |
|------|------|-------------|
| `avatar` | (String) | URL for the avatar image. |
| `thumbnail` | (String) | URL for the thumbnail image. |
| `video` | (String) | URL for the video thumbnail image. |
| `icon` | (String) | Icon to display in the list item. |
| `label` | (String) | Primary label text. |
| `subLabel` | (String) | Secondary label text. |
| `badge` | (Number) | Badge number to display. |
| `text` | (String) | Main text content of the list item. |
| `leadingCheckbox` | (Boolean) | Indicates if a leading checkbox is displayed. |
| `leadingRadioButton` | (Boolean) | Indicates if a leading radio button is displayed. |
| `leadingSwitch` | (Boolean) | Indicates if a leading switch is displayed. |
| `trailingCheckbox` | (Boolean) | Indicates if a trailing checkbox is displayed. |
| `trailingRadioButton` | (Boolean) | Indicates if a trailing radio button is displayed. |
| `trailingSwitch` | (Boolean) | Indicates if a trailing switch is displayed. |
| `selected` | (Boolean) | Indicates if the list item is selected. |
| `routerLink` | (String) | URL for routing purposes. |
