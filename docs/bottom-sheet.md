# MDBottomSheetComponent
The `MDBottomSheetComponent` interface represents a `md-bottom-sheet` element in the DOM. Represents a bottom sheet component that extends from MDSheetComponent.

## Inheritance
MDSheetComponent


## Instance properties
This interface also inherits properties from its parent, `MDSheetComponent`. 

name | type | desc
--- | --- | ---
variants | Read only | Variants for different styles of the bottom sheet.
