# MDIconButtonComponent

The `MDIconButtonComponent` interface represents a `<md-icon-button>` element in the DOM.
Represents an icon button component that extends MDComponent.



### Events

| Event | Description |
|-------|-------------|
| `MDIconButtonComponent#event:onIconButtonClick - Fired when the icon button is clicked.` | |




# properties


### Instance properties

| Name | Type | Description |
|------|------|-------------|
| `variant` | (String) | The variant style of the icon button (e.g., &quot;filled&quot;, &quot;tonal&quot;, &quot;outlined&quot;, &quot;toggle&quot;). |
| `icon` | (String) | The icon name or URL to be displayed on the button. |
| `selected` | (Boolean) | Indicates if the icon button is selected. |
| `disabled` | (Boolean) | Indicates if the icon button is disabled. |
