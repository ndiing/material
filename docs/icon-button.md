# MDIconButtonComponent
The `MDIconButtonComponent` interface represents a `md-icon-button` element in the DOM. Icon button component for Material Design.

## Inheritance
MDComponent

## Events
Listen to these events using `addEventListener()`, or by assigning an event listener to the `@onEventName` property of this interface.

name | desc
--- | ---
onIconButtonClick | Event fired when the icon button is clicked.


## Instance properties
This interface also inherits properties from its parent, `MDComponent`. undefined

name | type | desc
--- | --- | ---
variant | String | The variant style of the icon button (e.g., filled, tonal, outlined, toggle).
icon | String | The icon to display inside the icon button.
selected | Boolean | Indicates if the icon button is selected.
disabled | Boolean | Indicates if the icon button is disabled.
