# MDAttributeObserver
Class to observe changes to a specified attribute on a target element.

## Constructor
Creates an instance of MDAttributeObserver.

name | type | desc
--- | --- | --- 
callback | function | Callback function to be executed when the attribute changes.



## Instance methods
name | params | desc
--- | --- | ---
observe | target,attributeName | Starts observing changes to the specified attribute on the target element.
