# MDFabComponent

The `MDFabComponent` interface represents a `<md-fab>` element in the DOM.
Floating Action Button (FAB) component used for primary actions in an application.







# properties


### Instance properties

| Name | Type | Description |
|------|------|-------------|
| `variant` | (String) | Specifies the variant of the FAB. Possible values are &quot;small&quot;, &quot;large&quot;, &quot;surface&quot;, &quot;secondary&quot;, &quot;tertiary&quot;, &quot;unelevated&quot;, &quot;extended&quot;. |
| `icon` | (String) | Specifies the icon content of the FAB. |
| `label` | (String) | Specifies the label content of the FAB. |
| `selected` | (Boolean) | Indicates whether the FAB is selected. |
| `disabled` | (Boolean) | Indicates whether the FAB is disabled. |
