# MDDataTableNativeColumnComponent

The `MDDataTableNativeColumnComponent` interface represents a `<md-data-table-native-column>` element in the DOM. Represents a native column component for an MD data table.

## Instance Methods
This interface also inherits methods from its parent, `HTMLTableCellElement`.

| Name | Parameters | Description |
| --- | --- | --- |
| addController |  | Adds a controller for handling specific functionalities. |
