# MDRadioButtonComponent

The `MDRadioButtonComponent` interface represents a `<md-radio-button>` element in the DOM.
MDRadioButtonComponent is a custom radio button element with support for ripple effects.



### Events

| Event | Description |
|-------|-------------|
| `MDRadioButtonComponent#event:onRadioButtonNativeInput - Fired when the radio button&#x27;s native input value changes.` | |
| `MDRadioButtonComponent#event:onRadioButtonNativeReset - Fired when the radio button&#x27;s native input is reset.` | |




# properties


### Instance properties

| Name | Type | Description |
|------|------|-------------|
| `checked` | (Boolean) | Indicates whether the radio button is checked. |
| `defaultChecked` | (Boolean) | The default checked state of the radio button. |
| `disabled` | (Boolean) | Indicates whether the radio button is disabled. |
| `indeterminate` | (Boolean) | Indicates whether the radio button is in an indeterminate state. |
| `value` | (String) | The value associated with the radio button. |
| `name` | (String) | The name attribute for the radio button. |
| `form` | (Object) | The form element associated with the radio button. |
| `type` | (String) | The type of the radio button, typically &quot;radio&quot;. |
