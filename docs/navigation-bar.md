# MDNavigationBarComponent
The `MDNavigationBarComponent` interface represents a `md-navigation-bar` element in the DOM. Represents a navigation bar component that includes a tree view for navigation.

## Inheritance
MDSheetComponent


## Instance properties
This interface also inherits properties from its parent, `MDSheetComponent`. 

name | type | desc
--- | --- | ---
properties | Read only | Properties inherited from MDSheetComponent and MDTreeComponent.
body | Read only | Returns the body content of the navigation bar, which includes a tree component.
body | Read only | Sets the body content of the navigation bar.
