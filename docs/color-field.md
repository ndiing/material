# MDColorFieldComponent
The `MDColorFieldComponent` interface represents a `md-color-field` element in the DOM. MDColorFieldComponent provides a custom color input field with an integrated color picker. Extends MDTextFieldComponent to incorporate the basic text field functionality.

## Inheritance
MDTextFieldComponent


## Instance properties
This interface also inherits properties from its parent, `MDTextFieldComponent`. 

name | type | desc
--- | --- | ---
actions | Read only | Returns the actions available for the color field.
actions | Read only | Setter for actions, currently does nothing as actions are fixed.
