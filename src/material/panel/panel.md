# MdPanelScrimComponent (Panel)

`MdNavigation` is a utility class for handling placement in a web application.

_Panel is the visual position of elements on the screen_

- Destinations don't change. They should be consistent across app screens.

## Properties

| Property       | Type    | Default | Description                                         |
| -------------- | ------- | ------- | --------------------------------------------------- |
| label          | String  |         | The label text for the list item.                   |
| supportingText | String  |         | Supporting text for the list item.                  |
| badge          | String  |         | The badge text for the list item.                   |
| leadingItems   | Array   | [ ]     | An array of leading items for the list item.        |
| trailingItems  | Array   | [ ]     | An array of trailing items for the list item.       |
| activated      | Boolean | false   | Reflects whether the list item is activated or not. |
