# MdIconComponent

`MdIconComponent` is a custom LitElement representing a material design icon.

## Properties

No properties are defined in this component.

## Instance Methods

-   `dispatchCustomEvent()`: Dispatches a custom event named "custom-event".

## Events

-   `custom-event`: A custom event dispatched by the component.

## Examples

### Basic Usage

```html
<md-icon></md-icon>
```
