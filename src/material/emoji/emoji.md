# MdEmojiComponent (Emoji)

`MdEmojiComponent` is a custom LitElement representing a material design emoji.

_Emojis are small, digital icons used to convey emotions, expressions, or ideas in electronic communication. They have become an integral part of online conversations and are often used to add a visual and emotional element to text-based messages._

## Usage

Emojis have evolved into a universal language that transcends linguistic barriers and adds nuance to digital communication.

## Properties

No properties are defined in this component.

## Instance Methods

- `dispatchCustomEvent()`: Dispatches a custom event named "customEvent" with the detail "Event detail".

## Events

- `customEvent`: A custom event dispatched by the component. It includes a detail property with the value "Event detail".

## Examples

```html
<md-emoji></md-emoji>
```
