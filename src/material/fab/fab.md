# MdFabComponent

`MdFabComponent` is a custom LitElement representing a material design floating action button (FAB).

## Properties

-   `type` (String): Specifies the type of the button. Default value is "button".
-   `label` (String): The label text for the FAB.
-   `icon` (String): The icon to be displayed within the FAB.
-   `size` (String): The size of the FAB. Possible values are "small" and "large".
-   `extended` (Boolean): Indicates whether the FAB should have an extended style.

## Instance Methods

-   `fabNative`: Returns the native button element.

## Examples

### Basic Usage

```html
<md-fab></md-fab>
```
