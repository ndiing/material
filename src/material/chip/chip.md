# MdChip (Chip)

> _Chips help people enter information, make selections, filter content, or trigger actions_

- Use chips to show options for a specific context
- Four types: assist, filter, input, and suggestion
- Chip elevation defaults to 0 but can be elevated if they need more visual separation

## Usage

Chips help people enter information, make selections, filter content, or trigger actions. They're best used to help users accomplish their current task faster and easier.

There are four types of chips:

`1` Assist, `2` filter, `3` input, `4` suggestion

## Properties

| Property     | Type    | Default | Description                                          |
| ------------ | ------- | ------- | ---------------------------------------------------- |
| leadingIcon  | String  |         | Icon to be displayed before the chip label.          |
| avatar       | String  |         | URL of the avatar image to be displayed in the chip. |
| trailingIcon | String  |         | Icon to be displayed after the chip label.           |
| label        | String  | "Label" | The text content of the chip label.                  |
| activated    | Boolean | false   | Reflects whether the chip is in an activated state.  |
